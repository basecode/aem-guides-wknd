# WKND Sites Starter Kit

This is the Sites Starter Kit for the WKND Reference site: [https://www.wknd.site/](https://www.wknd.site/)

## What does the Starter Kit contain?

## Site Template

### Build

```
cd site.template
mvn clean install
```

## Site Theme

### Build

```
cd site.theme
npm install
npm run dev
```
